The Fluency Ceiling

If a student's decoding is not automatic, comprehension scaffolds alone cannot fix it. Working memory is conserved: every unit spent on decoding is a unit unavailable for meaning-making. This video shows the slosh — low fluency starves comprehension, high fluency frees it.
