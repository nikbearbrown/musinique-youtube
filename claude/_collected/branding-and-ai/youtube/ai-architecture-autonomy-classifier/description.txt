Build an AI Architecture Autonomy Classifier with Claude — part of Nina Harris' course Branding and AI.

Every AI system sits somewhere on an autonomy spectrum. This teardown classifies architectures across four patterns — human-in-loop, supervised, horizon-autonomous, fully autonomous — and rates locatability: can a human find and correct an error in the output chain? Compounding error math: a system with 90% per-step accuracy running 10 steps reaches only 35% end-to-end accuracy. At 40 steps: 1.5%. The ARCHITECTURAL WARNING fires on Pattern C (horizon-autonomous) + LOW locatability.

Prompt used in this video:
  claude "classify this AI system architecture:
    [ARCHITECTURE DESCRIPTION]
    Pattern A: human-in-loop | B: supervised |
    C: horizon-autonomous | D: fully autonomous
    Locatability: HIGH/MEDIUM/LOW
    Compounding error estimate at N steps (0.9^N)
    Fire ARCHITECTURAL WARNING if Pattern C + LOW locatability."
