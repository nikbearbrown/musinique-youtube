Build a Madison Multi-Agent Architecture Mapper with Claude — Nina Harris' Branding and AI course, Chapter 22.

A CLI tool that classifies any AI product against four agent meanings and five Madison roles, then outputs a failure-locatability assessment. When something goes wrong in a multi-agent system, you need to know which node failed. This mapper makes that possible.

"Multi-agent" means different things to different people. This tool forces precision about which kind you're actually building.

Made live in class using the Brutalist video system — a summary of the concepts discussed in today's session of INFO 7375: Branding and AI at Northeastern University.
