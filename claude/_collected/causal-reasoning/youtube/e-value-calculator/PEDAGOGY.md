# PEDAGOGY
Concept: E-value for sensitivity analysis — calibrating causal claim robustness.
Problem: how strong does an unmeasured confounder need to be to explain away a finding?
Question: how robust is a risk ratio of 2.1 to unmeasured confounding?
Answer: E-value ≈ 3.6 — a confounder would need 3.6-fold associations with both exposure and outcome to fully explain the effect.
