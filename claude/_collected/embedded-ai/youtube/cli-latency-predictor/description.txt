"How fast is my model?" has four answers, and only their sum ships. This video builds a latency predictor with Claude Code that decomposes inference into pipeline stages — weight load, compute, memory movement, overhead — and estimates each stage's wall-clock on a target. The dominant stage for memory-bound models is almost never compute.
