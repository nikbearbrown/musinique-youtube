# Fact Check

- Neutral Fe ground configuration is `[Ar] 3d6 4s2`; Fe2+ ground configuration is `[Ar] 3d6`.
- Madelung/Aufbau order is a useful empirical configuration mnemonic, not a universal fixed orbital-energy spectrum.
- Correction: neutral Fe configuration does not justify the blanket claim “4s is lower than 3d in neutral Fe.” Orbital energies are model- and occupation-dependent.
- Ionization energies are total-energy differences between many-electron states, including relaxation, not simply negatives of frozen one-electron orbital energies.
- The more diffuse 4s electrons are removed before 3d electrons in transition-metal cations such as Fe2+.
