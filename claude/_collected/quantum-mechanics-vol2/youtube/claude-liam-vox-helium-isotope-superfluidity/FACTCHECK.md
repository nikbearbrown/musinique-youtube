# Fact Check

- Helium-4's lambda point is about 2.17 K; helium-3 superfluid phases occur on the millikelvin scale and depend on pressure.
- A helium-4 atom contains an even number of fermionic constituents and is a boson; helium-3 contains an odd number and is a fermion.
- Helium-3 superfluidity is a paired-fermion phenomenon.
- Correction to the source card: liquid helium-4 is strongly interacting, its transition is not simply ideal-gas BEC, and superfluidity does not require every atom to occupy one lowest orbital.
- “1000x” is a rounded title-scale comparison, not a universal exact ratio.
