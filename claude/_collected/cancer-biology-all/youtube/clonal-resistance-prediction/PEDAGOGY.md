# PEDAGOGY — clonal-resistance-prediction

## Learning Objective
Understand that targeted therapy resistance is pre-existing (not induced), that clonal evolution can be mapped, and that ctDNA liquid biopsy offers a detection window before imaging progression.

## Core Concept
Tumor heterogeneity means drug-resistant subclones exist at diagnosis, below the detection threshold of standard sequencing. Targeted therapy eliminates sensitive clones, creating selective pressure that allows resistant subclones to expand. This is Darwinian selection, not mutation induction.

## EGFR Resistance Cascade
1. 1st/2nd-gen TKI → T790M gatekeeper mutation (55%) → osimertinib works
2. Osimertinib → C797S solvent-front mutation (20%) → no approved therapy
3. Bypass mechanisms: MET/HER2 amplification (20%) → no approved therapy
4. SCLC transformation (5%) → chemotherapy, no targeted option

## ctDNA Window Concept
ctDNA detects circulating tumor DNA fragments shed from tumor cells. Rising ctDNA before clinical/imaging progression provides a 3-6 month therapeutic window to switch therapy. This window is only useful if:
1. The test is ordered on the right schedule
2. The result is interpreted correctly
3. An effective next-line therapy exists

## Pedagogical Structure
- Establish that resistance is pre-existing (paradigm shift)
- Map the known resistance mechanisms with frequencies
- Introduce ctDNA as early detection tool
- Identify the three conditions required for ctDNA to improve outcomes
