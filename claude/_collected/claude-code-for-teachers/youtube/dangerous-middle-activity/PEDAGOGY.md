# PEDAGOGY — dangerous-middle-activity

## Concept
Dangerous middle classroom activity — three scenarios, three probes, three capacities — building the categorization muscle through doing.

## Core question
What makes a good dangerous-middle classroom scenario?

## Answer
The task reads as pure pattern work, the bug is invisible without the specific probe, and the probe is concrete enough that a student can run it in two minutes on a live Claude session.

## Teaching pattern
- B00: outcome — students find the bug in all three tasks
- B01: reframe — categorization before delegation
- B02-B03: generate three scenarios — different capacities, different bugs
- B04: verify each probe is specific and capacity-named
- B05-B06: extend to non-code — research synthesis scenario
- B07: one class period per scenario, live probe required
- B08: forward link to specification template

## Bloom's level
Applying → Evaluating: students can run a probe on a Claude output and name the supervisory capacity the probe exercises.

## Common misconception addressed
Teachers often design activities around identifying good Claude outputs. The video establishes that the dangerous middle requires identifying plausible-but-wrong outputs — the task is to find the bug that looks like a correct answer, not to rate a visibly wrong one.
