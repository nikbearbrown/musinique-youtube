# FACTCHECK — exponent-rules

**Topic**: Exponent Rules and the Proof that a⁰ = 1

## Claims verified

1. **Quotient rule: aᵐ ÷ aⁿ = aᵐ⁻ⁿ (a ≠ 0)** — Standard. Proved from the definition of positive integer exponents.

2. **Proof that a⁰ = 1**: a³ ÷ a³ = a^(3−3) = a⁰, and a³ ÷ a³ = 1, therefore a⁰ = 1 — Correct. This is the standard derivation from the quotient rule, valid for a ≠ 0.

3. **0⁰ is indeterminate** — Correct. In analysis, lim_{x→0⁺} x^x = 1 (often used in combinatorics), but as a limit 0⁰ is indeterminate form. The reel correctly flags this caveat.

4. **Product rule: aᵐ · aⁿ = aᵐ⁺ⁿ** — Standard, correct.

5. **Power rule: (aᵐ)ⁿ = aᵐⁿ** — Standard, correct.

6. **Negative exponent rule: a⁻ⁿ = 1/aⁿ (a ≠ 0)** — Standard, correct.

7. **Fractional exponent: a^(1/n) = ⁿ√a** — Standard, correct.

8. **Numerical examples**: 2³·2² = 2⁵ = 32 ✓; 2⁵÷2² = 2³ = 8 ✓; (2³)² = 2⁶ = 64 ✓; 7⁰ = 1 ✓; 2⁻³ = 1/8 ✓; 8^(1/3) = 2 ✓

## No issues found. All six rules and examples are correct.
