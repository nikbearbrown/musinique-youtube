# FACTCHECK — discriminant-three-stories

**Topic**: The Discriminant Δ = b² − 4ac

## Claims verified

1. **Discriminant formula: Δ = b² − 4ac** — Standard. Derived from completing the square on ax² + bx + c = 0.

2. **Δ > 0 → two distinct real roots** — Correct. The ± in the quadratic formula gives two different real values.

3. **Δ = 0 → one repeated real root** — Correct. x = −b/(2a), a single repeated root (perfect square).

4. **Δ < 0 → no real roots (complex conjugate pair)** — Correct. √(negative) is not real; roots are complex a ± bi.

5. **Case D>0: x²−3x+2**: Δ = 9 − 8 = 1 > 0. Correct. Roots: x = (3 ± 1)/2 = 2 or 1. Two real roots. ✓

6. **Case D=0: (x−1)² = x²−2x+1**: Δ = 4 − 4 = 0. Correct. One repeated root x = 1. ✓

7. **Case D<0: x²+x+1**: Δ = 1 − 4 = −3 < 0. Correct. No real roots. ✓

8. **Quadratic formula: x = (−b ± √(b²−4ac)) / 2a** — Standard, correct.

9. **The discriminant predicts root type before solving** — Correct pedagogical claim. Computing Δ first is a valid and common strategy.

## No issues found. All discriminant claims and examples are correct.
