# FACTCHECK — domain-restrictions

**Topic**: Domain Restrictions — Vertical Asymptotes vs Removable Holes

## Claims verified

1. **f(x) = 1/(x−2) has a vertical asymptote at x = 2** — Correct. The function is undefined at x = 2 and |f(x)| → ∞ as x → 2. Domain: (−∞, 2) ∪ (2, +∞).

2. **g(x) = (x²−4)/(x−2) has a removable discontinuity (hole) at x = 2** — Correct. x²−4 = (x+2)(x−2), so g(x) = (x+2)(x−2)/(x−2) = x+2 for x ≠ 2. At x = 2: g(2) = 0/0 (indeterminate), not defined.

3. **lim_{x→2} g(x) = 4** — Correct: lim_{x→2} (x+2) = 2+2 = 4. The hole is at the point (2, 4).

4. **The hole does not disappear when you simplify** — Correct. The domain of the original function g excludes x = 2 even though the simplified form x+2 is defined there. The simplification is only valid for x ≠ 2.

5. **Distinction: asymptote vs hole** — Correct. Asymptote: denominator is zero, numerator is nonzero → function diverges. Hole (removable discontinuity): both numerator and denominator share a common factor that vanishes at the same point.

6. **Domain notation (−∞, 2) ∪ (2, +∞)** — Correct interval notation for domain of f(x) = 1/(x−2).

## No issues found. All domain restriction claims are correct.
