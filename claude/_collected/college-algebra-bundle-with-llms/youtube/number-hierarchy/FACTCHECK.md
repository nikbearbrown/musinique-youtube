# FACTCHECK — number-hierarchy

**Topic**: Number Set Hierarchy (N ⊂ Z ⊂ Q ⊂ R ⊂ C)

## Claims verified

1. **ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ ⊂ ℂ** — Standard result in every college algebra and real analysis text. Each set is a proper subset of the next.

2. **Natural numbers {1, 2, 3, …}** — Standard convention (positive integers). Some texts include 0; this reel uses the traditional college algebra definition without 0, which is the majority convention in US textbooks.

3. **Integers {…, −2, −1, 0, 1, 2, …}** — Standard. Includes all positive and negative whole numbers plus zero.

4. **Rationals = {p/q | p, q ∈ ℤ, q ≠ 0}** — Standard definition. Every integer is rational (q=1).

5. **Reals include irrationals** — Correct. ℝ = ℚ ∪ (irrationals). π and √2 are standard examples of irrationals.

6. **π is irrational** — Proved by Lambert (1761). π cannot be written as p/q for any integers p, q. Correct.

7. **√(−1) = i is complex, not real** — Correct. i ∉ ℝ; it is the imaginary unit, element of ℂ = {a + bi | a, b ∈ ℝ}.

8. **3 belongs to all five sets** — Correct: 3 ∈ ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ ⊂ ℂ.

9. **−2 is an integer but not a natural number** — Correct under the {1, 2, 3, …} convention.

10. **½ is rational but not an integer** — Correct: ½ = 1/2, p=1, q=2.

## No issues found. All claims correct for standard college algebra.
