## What the learner already knows
Frequency, energy, the concept of a work function as an escape energy, basic wave/particle ideas.

## What this reel teaches
Light arrives as photons each carrying E=hν. A single photon acts alone — it either exceeds the work function threshold or not, regardless of intensity. Below threshold: zero electrons from any lamp. Above threshold: KE=hν−φ per electron, independent of brightness. This was Einstein's Nobel-winning insight.
