# PEDAGOGY — medhavy-vox-3d-matrices

## What the learner already knows
What a vector is, what a linear transformation preserves, î/ĵ/k̂ basis vectors

## What this reel teaches
Any linear transformation in 3D is fully determined by where the three basis vectors land — because every other vector is a weighted sum of them. Those nine landing numbers, arranged as columns, form the 3×3 matrix. Matrix-vector multiplication is the direct consequence of this linearity.

## Key facts
- Linear transformations preserve grid lines and origin ✓
- î=(1,0,0), ĵ=(0,1,0), k̂=(0,0,1); every vector is a weighted sum ✓
- 3×3 matrix columns = where î, ĵ, k̂ land ✓
- 90° Y-rotation: (2,−1,3) → (3,−1,−2) ✓
- N dimensions require N² numbers ✓

## Exclusions honored

- No matrix inversion
