## Claim-by-Claim Verification

| Beat | Claim | Verdict | Source / Note |
|------|-------|---------|---------------|
| B01 | Leukemia patient starts a new drug; blast count falls; cells do not die | ✓ | Chapter: "clinical responses include patients whose leukemic blasts differentiate into mature neutrophils — the disease resolving through differentiation rather than cytotoxicity" |
| B02 | Blood smear shows mature neutrophils with segmented nuclei in AML patient on IDH inhibitor | ✓ | Chapter: "leukemic blasts differentiate into mature neutrophils" |
| B03 | The mature neutrophils carry the original IDH1 mutation | ✓ | Chapter and card: "These neutrophils carry the original IDH1 mutation in their genome. The cancer cells did not die; they differentiated" |
| B04 | Question stated: blast counts fall but cells don't die, they mature; how does blocking a metabolic enzyme produce differentiation? | ✓ | Accurate framing of the mechanistic mystery |
| B05 | Normal blood development: stem cell matures through stages and dies of old age | ✓ | Standard hematology; implied by chapter context |
| B06 | In AML, blasts accumulate because the differentiation gate is locked | ✓ | Chapter: "an epigenetic block to differentiation" |
| B07 | IDH1 mutation produces 2-hydroxyglutarate (2HG); 2HG structurally resembles alpha-ketoglutarate which demethylases need | ✓ | Chapter: "Mutant IDH1 and IDH2 enzymes produce a metabolite, 2-hydroxyglutarate, that is structurally similar to alpha-ketoglutarate and competitively inhibits the TET demethylases and KDM histone demethylases" |
| B08 | 2HG jams demethylases; methyl marks on differentiation genes stay in place | ✓ | Chapter: "The result is hypermethylation of CpG islands and silencing of differentiation genes — an epigenetic block to differentiation" |
| B09 | Ivosidenib blocks mutant IDH1; 2HG falls; demethylases recover; methyl marks erased | ✓ | Chapter: "Block the mutant IDH enzyme, 2-HG falls, demethylase activity recovers, methylation patterns normalize, and leukemic cells differentiate" |
| B10 | The differentiation program was intact; only the chemical lock prevented it from running | ✓ | Chapter: "it is releasing an epigenetic lock" |
| B11 | Day 0: 80% blasts; Day 14: bilobed nuclei; Day 28: mature neutrophils; Day 56: <5% blasts | ILLUSTRATIVE | These specific percentages and timepoints are invented but plausible; the general pattern of progressive differentiation is supported by the chapter and card |
| B12 | Oncometabolite locked the developmental program; ivosidenib removed the lock; cancer matured | ✓ | Chapter: "A metabolic drug restored a developmental program that an oncometabolite had locked" (card) |
