# Vox-Style Explainers, Explained — a Vox explainer about itself

Sources: `aspects/explainer/vox-explainer/SKILL.md` (doctrine),
`docs/TUTORIAL.md` §3.8, `docs/pipeline.md`, `AGENTS.md`. 26 beats,
est. ~4:15. **Fully recursive:** every DOCUMENT beat shows THIS film's own
files; the slate (B15) and burn-in (B21) stills are frames harvested from this
film's own pass-1 cut. New playlist pilot: "Unreal Reels (Vox Style)".
Harvest order matters — own-source slots become grabbable as the build
progresses (see SHOTLIST). REVISED 2026-07-06, the interface decision:
conversation moments are Cowork captures of THIS conversation, machine
moments are live terminal recordings, B04 is Bear's own photo. Paid spend:
narration + ONE AI still (B18 — it must stay generated; it demonstrates the
STAND-IN X). B22 narration rewritten to match; audio regen `--only B22`.

**B03 · GRAPHIC** — the laundering funnel (mismatched plates → one treatment → one plane)

**B04 · COMPOSITE·own** — Bear's own saturated photo, treated + parameter chips

> Then the restraint that keeps it alive: two accent colors, serif labels
> with hairline underlines, one editor's pen per graphic. Decoration that
> isn't information gets deleted.

**B06 · DOCUMENT·own·annotate** — THIS beat_sheet.json, B06 entry highlighted

**B07 · GRAPHIC** — audio clock (this sheet's own measured durations as bars)

**B08 · GRAPHIC** — the two-axis grid (5 types × 3 sources)
