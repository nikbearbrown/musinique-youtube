Claude Opus wrote a complete C compiler in Rust — frontend, SSA IR, optimizer, code generator, assembler, linker, DWARF debug info — with zero human pair-programming. Four target architectures: x86-64, i686, AArch64, RISC-V. No external toolchain dependencies.

The methodology is the finding. One rule: humans write tests, Claude writes everything else — no hints, no debugging help. When Claude gets stuck, it reads the failing test and iterates. The constraint is the method.

The README's caveat: "I do not recommend you use this code — none of it has been validated for correctness." That caveat IS the point. A model that can build something it shouldn't be trusted to deploy is still a model that can build it.
