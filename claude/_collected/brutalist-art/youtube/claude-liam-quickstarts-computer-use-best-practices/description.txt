The computer-use demo is impressive. The production agent is radically different. The naive loop — screenshot, full-res image, tokens, action, repeat — can burn $50 in a single task.

Three fixes: resize screenshots to 1568px (same visual info, fraction of tokens), prune older screenshots from context (the model only needs recent state), and cache the system prompt (instructions don't change per step). Add batched tool calls to cut round-trips, and trajectory recording to log every action as a structured event you can replay and audit.

The gap between the demo and production is everything.
