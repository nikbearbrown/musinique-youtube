## Rounding note
The article text says "20%" overall agent adoption; the discipline-level figure
uses a 19% reference line (Fig 2). The chart uses 19% on the reference line
(figure value) and the narration/cold-open uses "one in five" (= 20%, text value).
Similarly the article rounds economics to "38%" in text; figure shows 38% — consistent.
