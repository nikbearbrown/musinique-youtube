# SOURCES — Claude's Values Across Models and Languages

## Primary source

Anthropic. "Claude's Values Across Models and Languages." July 13, 2026.
Published at anthropic.com/research.

## Dataset facts (from the paper)

- 700,000 conversations studied in prior work (value extraction phase)
- 3,307 distinct values identified in Claude's responses
- 309,815 real Claude.ai conversations used for dimensionality reduction
- 3 models × 20 languages × ~5,000 conversations per pair
- 4 principal axes extracted; capture 15% of variance
- 339 hand-clustered groups (intermediate compression step)

## Four axes (from the paper)

1. Deference ↔ Caution
2. Warmth ↔ Rigor  
3. Depth ↔ Brevity
4. Candor ↔ Execution
