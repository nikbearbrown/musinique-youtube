Some things are easier to show than to describe. Writing style is one of them. Few-shot prompting is implicit style encoding: the examples do more work per token than any instruction.

A well-chosen set of examples encodes output length, vocabulary register, structural template, domain jargon, and edge-case handling — all at once. Instruction expands these into five separate paragraphs. Examples compress them into three sentences.

Example quality matters more than count. One bad example contaminates the pattern. Three rules: representative inputs, internally consistent style, examples that match the target distribution.
