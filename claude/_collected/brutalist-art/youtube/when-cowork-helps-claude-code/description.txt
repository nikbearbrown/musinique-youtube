When Cowork Can Help Claude Code — video 3 of "Brutalist: Claude for Video Production."

A true story from making the last video. Claude Code was building the installs explainer and stalled on six terminal shots — not with an error, but with optimism: "just a couple more minutes," for the better part of an hour. It even blamed 43 stale Chrome processes that turned out to be a real browser and Spotify.

The fix wasn't smarter debugging. It was a second set of eyes, in a different seat, with nothing invested in the failing approach — which saw in one look that the build had almost succeeded and that a purpose-built helper was sitting right there, unused. Four commands, seconds, done.

The lesson: any single agent can tunnel, and when it does, another vantage breaks the loop — and you are the one who calls for it. That's conducting too.
