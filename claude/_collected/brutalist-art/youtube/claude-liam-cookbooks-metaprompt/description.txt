A metaprompt is a prompt that writes prompts. Feed it a task description and it returns a full production template: role, context, input format, instructions, output format, edge cases — all of it. The Anthropic metaprompt is the reference implementation.

You are not skipping prompt engineering. You are delegating the first draft to a system that has read the manual. The blank page is gone. The judgment is still yours.

Never ship the output without review. The metaprompt cannot know your production data schema, your specific edge cases, or the failures you've already seen. Generate, test on real inputs, iterate.
