## Utility-framing lint (every beat must be FOR the viewer, not about the tool)

- B01: viewer learns the folder workflow is real — the file that comes back is theirs
- B02: viewer won't be blindsided by "0 lines of code" — the code exists; look at it when it breaks
- B03: viewer takes away one concrete move: /skill-creator to save their way of working
- B04: viewer learns *where* the AskUserQuestion trick actually pays (Chat/Code, not Cowork)
- B05: viewer gets the right mental model for big folders — librarian, not scanner
- B06: viewer stops burning usage on High; sets the dial to Medium for everyday work
- B07: viewer leaves knowing the three omitted truths that change how they work

No beat reads as feature advertisement; two beats (B02, B06) actively push back on
the marketing. **PASS**
