# Fact Check

- `psi(x,t) = phi(x) exp(-iEt/hbar)` for an energy eigenstate.
- The phase rotates at angular frequency `E/hbar` in radians per second.
- Cycles per second would be `E/h`; the discarded draft incorrectly called `E/hbar` turns per second.
- `|psi|^2 = |phi|^2` because the phase cancels with its conjugate.
- A global phase is unobservable; relative phase in a multi-energy superposition can affect observables.
