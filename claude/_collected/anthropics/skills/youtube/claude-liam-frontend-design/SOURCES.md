## Key facts used

- 3 AI design defaults to avoid: (1) warm cream + high-contrast serif + terracotta accent; (2) near-black + acid-green or vermilion; (3) broadsheet + hairline rules + newspaper columns
- Design plan token system: 4–6 named hex values, typefaces per role (display + body + utility), layout concept with ASCII wireframes, signature element
- Two-pass process: brainstorm plan → critique (check for generic defaults) → build
- Hero as thesis: open with the most characteristic thing in the subject's world
- Typography carries personality: pair faces deliberately for this brief, not default choices
- Structure is information: numbered markers only if content is an actual sequence
- Motion serves the subject: orchestrated moments beat scattered effects
- Restraint: spend boldness in one place, Chanel rule (remove one accessory)
- Writing: active voice, errors explain + direct, one job per element, cohesive vocabulary
