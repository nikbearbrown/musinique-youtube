# Fact Check

- For orbital angular momentum, `L²|ℓ,m⟩ = ℓ(ℓ+1)ℏ²|ℓ,m⟩` and `Lz|ℓ,m⟩ = mℏ|ℓ,m⟩`, with integer `m = −ℓ,…,+ℓ`.
- With `L± = Lx ± iLy`, `L−L+ = L² − Lz² − ℏLz`, hence `L² = L−L+ + Lz² + ℏLz`.
- At the top rung, `L+|ℓ,ℓ⟩ = 0`, so the identity gives `ℓ²ℏ² + ℓℏ² = ℓ(ℓ+1)ℏ²`.
- In `|ℓ,ℓ⟩`, `⟨Lx⟩ = ⟨Ly⟩ = 0`, but `⟨Lx²+Ly²⟩ = ℓℏ²`.
- For `ℓ=2`, `L²=6ℏ²`, `Lz,max²=4ℏ²`, and five `m` values occur.
- The legacy claim that every atomic spectral line traces to this term was removed as an overstatement.
- Scope is orbital angular momentum; no spin, spherical-harmonic derivation, or representation theory.
