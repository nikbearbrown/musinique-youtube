# Fact check

- A Hahn echo uses a wait tau, a pi pulse, and a second wait tau; ideal static frequency offsets refocus at 2 tau.
- A transverse pi pulse maps rotating-frame phase +phi to -phi relative to the pulse axis; it does not change the frequency offsets.
- Static inhomogeneous dephasing contributes to T2-star and can be refocused; true/dynamic transverse loss is not generally undone.
- Echo decay can also include diffusion through gradients, time-dependent noise, and pulse imperfections, so it is not literally spin-spin interactions only.
- Data 0.85, 0.72, 0.52 at echo times 50, 100, 200 ms fit exp(-t_echo/T2) with T2 near 305 ms.
- A 10 rad/s static offset over 0.05 s accumulates 0.5 rad, canceled ideally in the second interval.
