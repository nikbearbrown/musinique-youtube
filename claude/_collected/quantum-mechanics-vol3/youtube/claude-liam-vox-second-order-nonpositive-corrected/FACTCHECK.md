# Factcheck

The build uses nondegenerate Rayleigh–Schrödinger perturbation theory for a unique unperturbed ground state and a Hermitian perturbation. In

`E_0^(2) = sum_(m != 0) |<m|H'|0>|^2 / (E_0-E_m)`,

each numerator is nonnegative and each denominator is negative, so the coefficient is nonpositive. It is not necessarily strictly negative: `H'` proportional to the identity makes all off-diagonal matrix elements vanish. The first-order coefficient can have either sign, so the full energy need not decrease. Degenerate ground spaces require degenerate perturbation theory. The hydrogen ground-state quadratic Stark shift is a valid scoped example.

The source card's invented particle-in-a-box coefficients and incorrect variational-principle connection are excluded.
