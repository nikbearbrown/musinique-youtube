# Pedagogy

Concrete-before-abstract sequence: wrong pinball model → translation symmetry → Bloch wave → symmetry breaking → relaxation time → filled-versus-partial band → superconductivity distinction → applied prediction.

The episode avoids the full proof of Bloch's theorem, translation-operator algebra, and band-index formalism. The viewer must leave able to state what periodicity removes, what it does not decide, and why a filled band changes the answer.
