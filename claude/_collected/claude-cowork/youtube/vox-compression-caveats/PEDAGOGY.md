# PEDAGOGY AUDIT — vox-compression-caveats

## One mechanism

The reel teaches exactly one mechanism: **compression is not neutral** — summarizing does not shrink a document evenly. Caveats, uncertainty, and minority findings read as peripheral, so they are the first thing cut. The result is confident-sounding prose produced from source material that was never that certain.
