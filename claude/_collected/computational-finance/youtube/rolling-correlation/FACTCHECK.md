# FACTCHECK — rolling-correlation

## Claims verified
- Rolling correlations: standard risk management tool ✓
- Cholesky decomposition for generating correlated returns ✓
- Crisis correlations spike toward 1.0 — well-documented stylized fact ✓
  (Longin & Solnik 2001; LTCM 1998; 2008 crisis)
- 60-day vs 120-day window tradeoff: shorter = more reactive, noisier ✓
- EWMA with lambda=0.94: RiskMetrics standard for daily data ✓
- Correlation stress test: regulatory standard for VaR stress testing ✓
- "Diversification fails in a crisis": consistent with empirical finance literature ✓
