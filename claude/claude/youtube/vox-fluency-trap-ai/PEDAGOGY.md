## Anchor-Not-Transcript Law
Beat B04 (CARD): card copy = "How does an AI produce a fake citation with the same confidence as a real one?" — narration expands: "A citation looks exactly like a real citation — author, journal, year, page range. It links to nothing. How does the AI produce it with the same confidence as a real one?" Card carries the question, narration carries the full framing. Different enough. PASS.

Beat B08 (CARD): card copy = "Fluency is not a verification signal." — narration says: "This is not a bug the next model will fix. It is structural." Card is the anchor phrase; narration is the explanation. PASS.

Beat B10 (CARD): card copy = "Style is not signal." — narration says: "The surface of the four good ones gave no signal about the two bad ones. Style is not signal. The appearance of a citation is not evidence that the source exists." Card is the shorter form; narration extends. PASS.

Beat B15 (CARD): card copy = "No verification path, no delegation." — narration: "Before you use an AI-generated citation, ask: how will I know if this source is real? If you cannot open it, it should not go in the deck." Card is the rule; narration is the application. PASS.
